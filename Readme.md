# Store API

## Descripción

Store API es una API desarrollada con Node.js, Express y MongoDB. Esta API está diseñada para gestionar operaciones de una tienda.

## Tecnologías

- Node.js
- Express
- MongoDB
- Mongoose
- dotenv
- axios
- cors
- express-validator
- multer
- nodemon (desarrollo)
